# Technology Stack - Healthcare & Telemedicine App

## Frontend
- **Framework**: React.js (Vite)
- **Styling**: Tailwind CSS
- **State Management**: Redux Toolkit
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Charts**: Recharts / Chart.js
- **RTK Query**: For efficient API fetching

## Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB (Mongoose)
- **Real-time**: Socket.io
- **WebRTC Signaling**: Socket.io
- **Auth**: JWT + Mock OTP logic

## AI & LLMs
- **LLM Provider**: Cohere
- **Framework**: LangChain / LangGraph
- **Use Cases**: 
  - AI Symptom Checker (Agentic mapping)
  - Prescription Reader (OCR + Extraction)
  - Lab Report Analysis (Computer Vision/Text Analysis)

## Infrastructure
- **Containerization**: Docker & Docker Compose
- **Payment**: Razorpay (Test Mode)
- **Environment**: Dotenv
